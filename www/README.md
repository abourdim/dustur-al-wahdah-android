# 🤝 Charter of Cultural Unity — دستور الوحدة الثقافية بين المسلمين

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **دستور الوحدة الثقافية بين المسلمين (Charter of Cultural Unity Among Muslims)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on Islamic cultural unity, diversity within the Ummah, shared values, eliminating sectarianism
- 6 deep reflections with Quranic evidence
- Daily habit tracker, self-assessment quiz, duas
- Trilingual: Arabic, English, French with RTL/LTR
- 3 themes (Ocean, Night, Sky), Narrator, PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.

## Disclaimer

I am not a scholar. This is a humble effort by a Muslim parent to share Sheikh al-Ghazali's wisdom interactively. Content derived from the book and trusted Islamic sources. This is not a fatwa.
